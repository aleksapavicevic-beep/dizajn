                   KORAK-PO-KORAK POČETAK
════════════════════════════════════════════════════════════════════

KORAK 1: Provjera Preduvjeta (5 min)
────────────────────────────────────
Trebam:
✅ Node.js v18+        → https://nodejs.org/
✅ npm (uključen)      → Automatic sa Node.js
✅ Git (optional)      → https://git-scm.com/

Provjera:
  node --version        (trebalo bi v18+)
  npm --version         (trebalo bi v9+)

KORAK 2: Navigacija u Projekt (1 min)
──────────────────────────────────────
  cd c:\Users\User\Desktop\DEMENTIA\dementia-companion

KORAK 3: Instalacija (5-10 min)
────────────────────────────────
  npm install

KORAK 4: Environment Setup (2 min)
──────────────────────────────────
  copy .env.local.example .env.local
  code .env.local          (otvori sa VS Code)

  Za testiranje: Ostaviti prazno! Aplikacija koristi mock podatke.
  Za produkciju: Vidi COMPLETE_SETUP_GUIDE.md

KORAK 5: Pokretanje (1 min)
──────────────────────────
  npm start
  
  Trebalo bi vidjeti:
  ┌─────────────────────────────────┐
  │ Use the app UI to navigate to   │
  │ npm run android                 │
  │ npm run ios                     │
  │ npm run web                     │
  │ or press 'a/i/w' in this window│
  └─────────────────────────────────┘

KORAK 6: Testiranje (2 min)
───────────────────────────
  iPhone:     Pritisnuti 'i'
  Android:    Pritisnuti 'a'
  Web:        Pritisnuti 'w'
  
  ILI Expo Go:
  1. Preuzmi Expo Go sa App Store / Google Play
  2. Skeniraj QR kod sa telefona
  3. Aplikacija se otvara! 🎉
